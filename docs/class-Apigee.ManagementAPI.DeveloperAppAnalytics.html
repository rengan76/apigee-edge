Parameters

$devId
The ID of the developer who owns the app.
$appName
The name of the app.
$metric

A value of 'message_count', 'message_count-first24hrs', 'message_count-second24hrs', 'error_count', 'error_count-first24hrs', 'user_count', 'user_count-first24hrs', 'total_response_time', 'max_response_time', or 'min_response_time'.

$tStart

Time start, expressed as:

  • UNIX timestamp
  • mm/dd/YYYY hh:ii
  • Any other format that the underlying strtotime() PHP function can parse. See http://php.net/strtotime. It parses them out to a UNIX timestamp if possible, otherwise it throws an exception.
$tEnd

Time end, expressed as:

  • UNIX timestamp
  • mm/dd/YYYY hh:ii
  • Any other format that the underlying strtotime() PHP function can parse. See http://php.net/strtotime. It parses them out to a UNIX timestamp if possible, otherwise it throws an exception.
$tUnit

Time unit: a value of 'second', 'minute', 'hour', 'day', 'week', 'month', 'quarter', or 'year'.

$sortBy
A comma separated list of the same values as $metric.
$sortOrder
Either 'ASC' or 'DESC'.
