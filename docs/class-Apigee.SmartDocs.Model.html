public integer
# importFile( string $document, string $document_format, string $content_type, null|string $modelId = null )

Import a model from a file.

Import a model from a file.

The file is passed into the method as a string. Note that you do not have to create a revision first, this method will automagically create the revision for you.

Note that Swagger 1.2 cannot be expressed as in a single-file format, so importFile can only be used with Swagger 2.0. For Swagger 1.2, use importUrl() instead.

Parameters

$document
The text to import into the model.
$document_format

The format, either 'wadl', 'swagger', or 'apimodel'.

$content_type

is the mime type, which valid values depend on the document format: wadl: 'application/xml' swagger: 'application/yaml' or 'application/json' apimodel: 'application/json'

$modelId

The model id, if not passed will be the modelId from this object.

Returns

integer
Revision number of newly created revision.

Throws

Apigee\Exceptions\ParameterException
